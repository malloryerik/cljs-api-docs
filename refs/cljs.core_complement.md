(__complement__ f)<br>
</samp>

---

Takes a function `f` and returns a function that takes the same arguments as
`f`, has the same effects, if any, and returns the opposite truth value.

---

###### Examples:

```clj
(def a 10)
((complement #(= a %)) 12)
;;=> true
```

Source code @ [github](https://github.com/clojure/clojurescript/blob/r927/src/cljs/cljs/core.cljs#L1443-L1451):

```clj
(defn complement
  [f] 
  (fn 
    ([] (not (f)))
    ([x] (not (f x)))
    ([x y] (not (f x y)))
    ([x y & zs] (not (apply f x y zs)))))
```
