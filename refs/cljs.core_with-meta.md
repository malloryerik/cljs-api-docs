 <samp>
(__with-meta__ obj m)<br>
</samp>

---

Returns an object of the same type and value as `obj`, with map `m` as its
metadata.

---

###### Examples:

```clj
(def a ^:foo [1 2 3])
(def b (with-meta a {:bar true}))

(= a b)
;;=> true

(meta a)
;;=> {:foo true}

(meta b)
;;=> {:bar true}
```

```clj
(defn with-meta
  [o meta]
  (if (and (fn? o) (not (satisfies? IWithMeta o)))
    (MetaFn. o meta)
    (when-not (nil? o)
      (-with-meta o meta))))
```
