Source docstring:

```
Returns the map with the keys in kmap renamed to the vals in kmap
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r2067/src/cljs/clojure/set.cljs#L72-L81):

```clj
(defn rename-keys
  [map kmap]
    (reduce 
     (fn [m [old new]]
       (if (and (not= old new)
                (contains? m old))
         (-> m (assoc new (get m old)) (dissoc old))
         m)) 
     map kmap))
```
