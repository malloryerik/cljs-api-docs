Source docstring:

```
Non-atomically swaps the value of the volatile as if:
(apply f current-value-of-vol args). Returns the value that
was swapped in.
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r3117/src/clj/cljs/core.clj#L1965-L1970):

```clj
(defmacro vswap!
  [vol f & args]
  `(-vreset! ~vol (~f (-deref ~vol) ~@args)))
```
