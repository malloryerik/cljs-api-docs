 <samp>
(__last__ coll)<br>
</samp>

---

Returns the last item in `coll` in linear time.

`peek` is much faster than `last` for a vector.

---

###### Examples:

```clj
(last [1 2 3])
;;=> 3

(last [1 2])
;;=> 2

(last [1])
;;=> 1

(last [])
;;=> nil
```

```clj
(defn last
  [s]
  (let [sn (next s)]
    (if-not (nil? sn)
      (recur sn)
      (first s))))
```
