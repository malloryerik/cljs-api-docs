Source code @ [github](https://github.com/clojure/clojurescript/blob/r2069/src/cljs/clojure/zip.cljs#L157-L163):

```clj
(defn leftmost
  [loc]
    (let [[node {l :l r :r :as path}] loc]
      (if (and path (seq l))
        (with-meta [(first l) (assoc path :l [] :r (concat (rest l) [node] r))] (meta loc))
        loc)))
```
