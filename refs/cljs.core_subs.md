---

Returns the substring of `s` beginning at `start` inclusive, and ending at `end`
exclusive.

`end` defaults to the length of the string.

---




Source docstring:

```
Returns the substring of s beginning at start inclusive, and ending
at end (defaults to length of string), exclusive.
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r2024/src/cljs/cljs/core.cljs#L1762-L1766):

```clj
(defn subs
  ([s start] (.substring s start))
  ([s start end] (.substring s start end)))
```
