```
Adds x to the transient collection, and return coll. The 'addition'
may happen at different 'places' depending on the concrete type.
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r2411/src/cljs/cljs/core.cljs#L2846-L2857):

```clj
(defn conj!
  ([] (transient []))
  ([coll] coll)
  ([tcoll val]
    (-conj! tcoll val))
  ([tcoll val & vals]
    (let [ntcoll (-conj! tcoll val)]
      (if vals
        (recur ntcoll (first vals) (next vals))
        ntcoll))))
```
