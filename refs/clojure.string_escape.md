---

Return a new string, using `cmap` to escape each character `ch` from `s` as follows:

If `(cmap ch)` is nil, append ch to the new string.

If `(cmap ch)` is non-nil, append `(str (cmap ch))` instead.

Source code @ [github](https://github.com/clojure/clojurescript/blob/r3123/src/cljs/clojure/string.cljs#L179-L195):

```clj
(defn escape
  [s cmap]
  (let [buffer (StringBuffer.)
        length (.-length s)]
    (loop [index 0]
      (if (= length index)
        (. buffer (toString))
        (let [ch (.charAt s index)]
          (if-let [replacement (get cmap ch)]
            (.append buffer (str replacement))
            (.append buffer ch))
          (recur (inc index)))))))
```
