```
Evaluate a JavaScript string in the Node REPL process.
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r3148/src/clj/cljs/repl/node.clj#L51-L66):

```clj
(defn node-eval
  [repl-env js]
  (let [{:keys [in out]} @(:socket repl-env)]
    ;; escape backslash for Node.js under Windows
    (write out js)
    (let [result (json/read-str
                   (read-response in) :key-fn keyword)]
      (condp = (:status result)
        "success"
        {:status :success
         :value (:value result)}

        "exception"
        {:status :exception
         :value (:value result)}))))
```
