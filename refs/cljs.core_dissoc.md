---

dissoc(iate)

Returns a new map that does not contain a mapping for key(s).

Has no effect on the map type (hashed/sorted).

---

###### Examples:

```clj
(dissoc {:key "value" :key2 "value2"} :key)
;;=> {:key2 "value2"}
```

```clj
(defn dissoc
  ([coll] coll)
  ([coll k]
     (-dissoc coll k))
  ([coll k & ks]
     (let [ret (dissoc coll k)]
       (if ks
         (recur ret (first ks) (next ks))
         ret))))
```
