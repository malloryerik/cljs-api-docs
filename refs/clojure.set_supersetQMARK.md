 <samp>
(__superset?__ a b)<br>
</samp>

---

Returns true if `a` is a superset of `b`, false otherwise.

In other words, returns true if `a` contains all the elements of `b`.

Source docstring:

```
Is set1 a superset of set2?
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r1913/src/cljs/clojure/set.cljs#L138-L142):

```clj
(defn superset? 
  [set1 set2]
  (and (>= (count set1) (count set2))
       (every? #(contains? set1 %) set2)))
```
