(__empty__ coll)<br>

---

Returns an empty collection of the same category as `coll`.

Returns nil if `coll` is nil.

---

Source code @ [github](https://github.com/clojure/clojurescript/blob/r3153/src/cljs/cljs/core.cljs#L1437-L1441):

```clj
(defn empty
  [coll]
  (when-not (nil? coll)
    (-empty coll)))
```
