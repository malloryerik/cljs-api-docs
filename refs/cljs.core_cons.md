 <samp>
(__cons__ x coll)<br>
</samp>

---

Returns a new sequence where `x` is the first element and `coll` is the rest.

---

###### Examples:

```clj
(cons 1 (list 1 2 3))
;;=> (1 1 2 3)

(cons 1 [1 2 3])
;;=> (1 1 2 3)

(cons 1 nil)
;;=> (1)

(cons nil nil)
;;=> (nil)
```

Source code @ [github](https://github.com/clojure/clojurescript/blob/r1859/src/cljs/cljs/core.cljs#L2033-L2039):

```clj
(defn cons
  [x coll]
  (if (or (nil? coll)
          (satisfies? ISeq coll false))
    (Cons. nil x coll nil)
    (Cons. nil x (seq coll) nil)))
```
