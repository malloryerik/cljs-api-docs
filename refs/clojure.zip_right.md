

Source code @ [github](https://github.com/clojure/clojurescript/blob/r2356/src/cljs/clojure/zip.cljs#L135-L140):

```clj
(defn right
  [loc]
    (let [[node {l :l  [r & rnext :as rs] :r :as path}] loc]
      (when (and path rs)
        (with-meta [r (assoc path :l (conj l node) :r rnext)] (meta loc)))))
```
