Source code @ [github](https://github.com/clojure/clojurescript/blob/r3264/src/main/cljs/clojure/zip.cljs#L142-L148):

```clj
(defn rightmost
  [loc]
    (let [[node {l :l r :r :as path}] loc]
      (if (and path r)
        (with-meta [(last r) (assoc path :l (apply conj l node (butlast r)) :r nil)] (meta loc))
        loc)))
```
