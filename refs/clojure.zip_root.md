Source code @ [github](https://github.com/clojure/clojurescript/blob/r2024/src/cljs/clojure/zip.cljs#L124-L133):

```clj
(defn root
  [loc]
    (if (= :end (loc 1))
      (node loc)
      (let [p (up loc)]
        (if p
          (recur p)
          (node loc)))))
```
