Source docstring:

```
Returns a zipper for xml elements (as from xml/parse),
given a root element
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r2723/src/cljs/clojure/zip.cljs#L50-L58):

```clj
(defn xml-zip
  [root]
    (zipper (complement string?) 
            (comp seq :content)
            (fn [node children]
              (assoc node :content (and children (apply vector children))))
            root))
```
