Returns true if `x` is a sequence, false otherwise.

All collections can be converted into a sequence using `seq`.

Source code @ [github](https://github.com/clojure/clojurescript/blob/r1934/src/cljs/cljs/core.cljs#L1175-L1180):

```clj
(defn ^boolean seq?
  [s]
  (if (nil? s)
    false
    (satisfies? ISeq s)))
```
