conj(oin)

Returns a new collection with the `x`s "added" to `coll`.

The "addition" may happen at different "places" depending on the collection
type.

`(conj nil item)` returns `(item)`.

---

###### Examples:

Append a vector:

```clj
(conj [1 2 3] 4)
;;=> [1 2 3 4]
```

Prepend a list:

```clj
(conj (list 1 2 3) 0)
;;=> (0 1 2 3)
```

Prepend a sequence:

```clj
(def x (range 1 4))
;;=> (1 2 3)

(conj x 0)
;;=> (0 1 2 3)
```

Add to set:

```clj
(conj #{"a" "b" "c"} "d")
;;=> #{"a" "b" "c" "d"}
```

```clj
(defn conj
  ([coll x]
    (if-not (nil? coll)
      (-conj coll x)
      (list x)))
  ([coll x & xs]
    (if xs
      (recur (conj coll x) (first xs) (next xs))
      (conj coll x))))
```
