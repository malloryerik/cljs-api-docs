 <samp>
(__next__ coll)<br>
</samp>

---

Returns a sequence of the items after the first and calls `seq` on its argument.

Returns nil if `coll` is empty.

---

###### Examples:

```clj
(next [1 2 3])
;;=> (2 3)

(next [1 2])
;;=> (2)

(next [1])
;;=> nil

(next [])
;;=> nil
```

Source code @ [github](https://github.com/clojure/clojurescript/blob/r2060/src/cljs/cljs/core.cljs#L502-L509):

```clj
(defn ^seq next
  [coll]
  (when-not (nil? coll)
    (if (implements? INext coll)
      (-next ^not-native coll)
      (seq (rest coll)))))
```
