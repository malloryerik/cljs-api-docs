Returns true if `n` is an even number.

Throws an exception if `n` is not an integer.

Source code @ [github](https://github.com/clojure/clojurescript/blob/r2644/src/cljs/cljs/core.cljs#L3246-L3250):

```clj
(defn ^boolean even?
   [n] (if (integer? n)
        (zero? (bit-and n 1))
        (throw (js/Error. (str "Argument must be an integer: " n)))))
```
