```
Defines a scope where JavaScript's implicit "this" is bound to the name provided.
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r1909/src/clj/cljs/core.clj#L550-L554):

```clj
(defmacro this-as
  [name & body]
  `(let [~name (js-this)]
     ~@body))
```
