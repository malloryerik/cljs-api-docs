Source docstring:

```
Returns the loc of the leftmost child of the node at this loc, or
nil if no children
```


Source code @ [github](https://github.com/clojure/clojurescript/blob/r3195/src/cljs/clojure/zip.cljs#L98-L109):

```clj
(defn down
  [loc]
    (when (branch? loc)
      (let [[node path] loc
            [c & cnext :as cs] (children loc)]
        (when cs
          (with-meta [c {:l [] 
                         :pnodes (if path (conj (:pnodes path) node) [node]) 
                         :ppath path 
                         :r cnext}] (meta loc))))))
```
