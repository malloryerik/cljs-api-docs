Source code @ [github](https://github.com/clojure/clojurescript/blob/r1443/src/cljs/cljs/core.cljs#L876-L881):

```clj
(defn ^boolean set?
  [x]
  (if (nil? x)
    false
    (satisfies? ISet x)))
```
