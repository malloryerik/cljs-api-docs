 <samp>
(__butlast__ s)<br>
</samp>

---

Returns a sequence of all but the last item in `s`.

`butlast` runs in linear time.

---

###### Examples:

```clj
(butlast [1 2 3])
;;=> (1 2)

(butlast [1 2])
;;=> (1)

(butlast [1])
;;=> nil

(butlast [])
;;=> nil
```

```clj
(defn butlast [s]
  (loop [ret [] s s]
    (if (next s)
      (recur (conj ret (first s)) (next s))
      (seq ret))))
```
