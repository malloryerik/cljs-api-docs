Source code @ [github](https://github.com/clojure/clojurescript/blob/r3291/src/main/cljs/clojure/zip.cljs#L174-L181):

```clj
(defn insert-right
  [loc item]
    (let [[node {r :r :as path}] loc]
      (if (nil? path)
        (throw "Insert at top")
        (with-meta [node (assoc path :r (cons item r) :changed? true)] (meta loc)))))
```
